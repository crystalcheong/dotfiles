# Doom Core Keys (Week 1)

- `SPC f f`: find file
- `SPC p p`: switch project
- `SPC b b`: switch buffer
- `SPC /`: search in project
- `SPC w v` / `SPC w s`: split windows
- `SPC w h/j/k/l`: move between windows
- `SPC h k`: describe key
- `SPC h f`: describe function

## Git (Terminal-first)

- Terminal popup: `SPC o t`
- Terminal here (current dir): `SPC o T`
- Magit status: `SPC g g`
- Full cheatsheet: see `MAGIT.md`

## Devcontainers (Local edit + container run)

- `SPC o u`: `devcontainer up` for current workspace
- `SPC o e`: run arbitrary `devcontainer exec ...`
- `SPC o s`: open shell command in container context
